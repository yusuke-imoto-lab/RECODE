RECODE: Resolution of Curse of Dimensionality in Single-cell Data Analysis
====================================
**Re**\ solution of **c**\ urse **o**\ f **d**\ im\ **e**\ nsionality in **s**\ ingle-**c**\ ell data analysis (RECODE) is a noise reduction method for single-cell sequencing data. `github <https://github.com/yusuke-imoto-lab/screcode>`_

Installation
====================================
RECODE package supports PyPI install. 

.. code-block:: bash

 $ pip install screcode



You can also install the development version of lingam package from GitHub:

.. code-block:: bash

	$ pip install git+https://github.com/yusuke-imoto-lab/RECODE.git


To use RECODE, import ``screcode``. 

.. code-block:: python

 import screcode
