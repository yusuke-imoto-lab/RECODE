class screcode.RECODE(fast_algorithm=True, fast_algorithm_ell_ub=1000, seq_target='RNA', version=1, solver='auto', downsampling_rate=0.2, decimals=5, RECODE_key='RECODE', anndata_key='layers', random_state=0, verbose=True)ÔÉÅ

Bases: object

RECODE (Resolution of curse of dimensionality in single-cell data analysis). A noise reduction method for single-cell sequencing data.

Parameters:

  • fast_algorithm (boolean, default=True) ‚Äì If True, the fast algorithm is conducted. The upper bound of parameter \(\ell\) is set by fast_algorithm_ell_ub.

  • fast_algorithm_ell_ub (int, default=1000) ‚Äì Upper bound of parameter \(\ell\) for the fast algorithm. Must be of range [1,:math:infity).

  • seq_target ({'RNA','ATAC','Hi-C','Multiome'}, default='RNA') ‚Äì Sequencing target. If ‚ÄòATAC‚Äô, the preprocessing (odd-even stabilization) will be performed before the regular algorithm.

  • version (int default='1') ‚Äì Version of RECODE.

  • solver ({'auto', 'full', 'randomized'}, default="auto") ‚Äì

    If auto:

    set solver='randomized' if the number of samples (cells) are larger than 20,000. Otherwise set solver='full'.

    If full:

    run learning process using the full input matrix.

    If randomized:

    run learning process involving computing SVD and estimating the essential dimension using downsampled data with the rate downsampling_rate.

  • downsampling_rate (float, default=1000) ‚Äì Downsampling rate, which is only relevant when solver='randomized'.

  • decimals (int default='5') ‚Äì Number of decimals for round processed matrices.

  • RECODE_key (string, default='RECODE') ‚Äì Key name of anndata to store the output.

  • anndata_key ({'layers','obsm'}, default='layers') ‚Äì Attribute of anndata where the output stored.

  • random_state (int or None, default=0) ‚Äì Used when the ‚Äòrandomized‚Äô solver is used.

  • verbose (boolean, default=True) ‚Äì If False, all running messages are not displayed.

lognormalize(X, base=None, target_sum=10000.0, key=None)ÔÉÅ

Standard normalization: Normalize counts per cell and then logarithmize it: \(x_{ij}^{\rm log} = \log(x_{ij}^{\rm norm} + 1)\), where \(x_{ij}^{\rm norm} = c*x_{ij}/\sum_{i}x_{ij}\) and :math:`x_{ij}‚Äô is the count calue of :math:`i‚Äôth cell and :math:`j‚Äôth gene.

Parameters:

  • X (ndarray or anndata of shape (n_samples, n_features).) ‚Äì single-cell sequencing data matrix (row:cell, culumn:gene/peak).

  • base (positive number or None, default=None) ‚Äì Base of the logarithm. If None, natural logarithm is used.

  • target_sum (float, default=1e4,) ‚Äì Total value after count normalization, corresponding the coefficient \(c\) above.

  • RECODE_key (string, default=None) ‚Äì Key name of anndata to store the output. If None, the RECODE_key that is set initially is used.

class screcode.RECODE_core(method='variance', variance_estimate=True, fast_algorithm=True, fast_algorithm_ell_ub=1000, ell_manual=10, ell_min=3, version=1, random_state=0, verbose=True)ÔÉÅ

Bases: object

The core part of RECODE (for non-randam sampling data).

Parameters:

  • method ({'variance','manual'}) ‚Äì If ‚Äòvariance‚Äô, regular variance-based algorithm. If ‚Äòmanual‚Äô, parameter \(\ell\), which identifies essential and noise parts in the PCA space, is manually set. The manual parameter is given by ell_manual.

  • variance_estimate (boolean, default=True) ‚Äì If True and method='variance', the parameter estimation method will be done.

  • fast_algorithm (boolean, default=True) ‚Äì If True, the fast algorithm is done. The upper bound of essential dimension \(\ell\) is set in fast_algorithm_ell_ub.

  • fast_algorithm_ell_ub (int, default=1000) ‚Äì Upper bound of parameter \(\ell\) for the fast algorithm. Must be of range [1, infinity).

  • ell_manual (int, default=10) ‚Äì Manual essential dimension \(\ell\) computed by method='manual'. Must be of range [1, infinity).

  • ell_min (int, default=3) ‚Äì Minimam value of essential dimension \(\ell\)

  • version (int default='1') ‚Äì Version of RECODE.
